# Artist Management A&R - Professional Website

A modern, professional website for Artist Management A&R - a premier artist management and A&R company dedicated to discovering, developing, and managing exceptional musical talent.

## Structure

- **index.html** - Home page featuring company overview and featured artists
- **about.html** - About Us page with company story and expertise
- **projects.html** - Artist roster and portfolio showcase
- **skills.html** - Services offered (A&R, management, marketing, etc.)
- **contact.html** - Contact form for artist submissions and business inquiries

## Local Testing

To view the site locally:

1. Open `index.html` in your web browser, or
2. Run a local server:
   ```bash
   python3 -m http.server 8000
   ```
   Then navigate to `http://localhost:8000`

## Technologies

- HTML5
- CSS3 (custom responsive stylesheet)
- Font Awesome icons
- JavaScript for interactive enhancements

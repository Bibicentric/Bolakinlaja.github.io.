This site is automatically deployed to GitHub Pages using GitHub Actions.

### Automatic Deployment

Every push to the `main` or `master` branch triggers automatic deployment via the GitHub Actions workflow (`.github/workflows/pages.yml`).

### Manual Deployment

You can also trigger a manual deployment from the GitHub Actions tab.

### GitHub Pages Settings

1. Go to repository Settings > Pages
2. Source should be set to "GitHub Actions"
3. The site will be available at `https://bibicentric.github.io/Bolakinlaja.github.io./`
